# 📌 Projects Included

A desktop application that allows users to merge multiple PDF files into a single PDF using an intuitive graphical interface.

### Features

* Merge multiple PDF files
* Browse and select PDF documents
* Save merged PDF to a custom location
* Simple and user-friendly GUI

**Technologies Used**

* Python
* Tkinter
* PyPDF
* OS Module

# 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/Python-Mini-Projects.git
```

2. Navigate to the project directory.

```bash
cd Python-Mini-Projects
```

3. Open the desired project folder.

4. Install required dependencies (if needed).

```bash
pip install -r requirements.txt
```

5. Run the Python file.

```bash
python main.py
```
